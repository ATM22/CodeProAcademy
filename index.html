Python

Es un lenguaje de programación interpretado, fácil de aprender, con una sintaxis única que se centra en la legibilidad y cuenta con una gran biblioteca estándar que contiene códigos reutilizables para casi cualquier tarea.

C++

Es un lenguaje de programación de alto rendimiento, orientado a objetos y con una sintaxis similar a C, que se utiliza principalmente para el desarrollo de sistemas operativos, juegos y aplicaciones de alto rendimiento

Java

Es un lenguaje de programación orientado a objetos, con una sintaxis similar a C++, que se utiliza principalmente para el desarrollo de aplicaciones empresariales y de servidores.
