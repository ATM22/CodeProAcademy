PROGRAMACIÓN EN PYTHON

Python es un lenguaje de programación de alto nivel ampliamente utilizado en aplicaciones web, desarrollo de software, ciencia de datos, machine learning y otros campos en auge. A diferencia de otros lenguajes como Java o .NET, Python es interpretado, lo que significa que no es necesario compilarlo para ejecutar las aplicaciones escritas en Python. Es un lenguaje sencillo de leer y escribir debido a su alta similitud con el lenguaje humano. Además, es multiplataforma, de código abierto y gratuito, lo que permite desarrollar software sin límites. Con el paso del tiempo, Python ha ganado adeptos gracias a su sencillez y a sus amplias posibilidades, sobre todo en los últimos años, ya que facilita trabajar con inteligencia artificial, big data, machine learning y data science

El origen de Python se remonta a finales de los años 80, concretamente en 1989, cuando el programador holandés Guido Van Rossum comenzó a crear un nuevo lenguaje de programación. La necesidad personal de disponer de un lenguaje sencillo de leer, escribir y soportar fue el principal motivo que impulsó a Van Rossum a crear Python. Entre los objetivos principales de Guido estaba crear un lenguaje que obtuviera una sintaxis simple y limpia, con el que los programadores pudieran escribir el código de una forma clara, concisa y seguros de lo que estaban haciendo. En un principio, recibió influencia de ciertos lenguajes como ABC, Moldua-3 y C++. En 1991 tuvo lugar el lanzamiento de la primera versión de Python, la 0.9.0. Esto significó un antes y un después en el sector de la programación, ya que comenzaron a realizarse mejoras continuas por parte de Guido y otros profesionales. En el constante avance de Python se ha trabajado para incluir en él ciertas modernidades, la programación orientada a objetos de la que se disfruta hoy en día y la amplia biblioteca que lo caracteriza

Lección 1: Asignación de Variables

En esta lección, aprenderás que las variables en Python son espacios en la memoria donde se almacenan y recuperan datos, habiendo dos tipos principales: números y texto. También aprenderás a cómo declarar una variable y cómo Python identifica el tipo de datos almacenados. El video analiza la diferenciación de variables según la distinción entre mayúsculas y minúsculas, el proceso de ejecución y lo que se imprime en la pantalla cuando se ejecuta un programa en Python. Además, se enfatiza la simplicidad del uso de variables en Python.
